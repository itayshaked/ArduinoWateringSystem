## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

Server side:
* npm
  ```sh
  npm i node node-fetch cors express aws-sdk dotenv 
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/itayshaked/ArduinoWateringSystem.git
   ```
2. Install NPM packages
   ```sh
   npm i node node-fetch cors express aws-sdk dotenv
3. Run server
   ```sh 
   npm run start
4. Run client
    ```sh
    cd client && npm start
    ```
